Anchor Elements

An anchor Element can be used by adding an a tag. This tag should make a particular bit of text clickable. Once given a href attribute, where the href attribute links us to any particular place, we can click on text, and be transported to a given destination. There are further things that we add on besides the href, one of them is target="_blank", this tells the computer to open the image on a lank tab, otherwise the default is target="_self" Another one which is used is rel="noopener noreferrer", which states that the relation between your page will be one where the noopener prevents the opened link from gaining access to the webpage from which it is opened. The noreferrer value prevents the link from knowing which website has a reference to the link. These added measures are for security.

Images

The image element, is made of img tag. The img element takes in an attribute, of src. This will take in the file path for a particular image that you might want to use.
